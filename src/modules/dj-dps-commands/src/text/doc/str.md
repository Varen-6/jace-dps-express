# Text.str

**Converts script context** to `string`

### Aliases

They are follows aliases of command name:

- `str`
- `text`
- `string`

### How to use

```dps
<?json
    [1,1,1]
?>

str()
json()
str()
```

This data processing script returns

```dps
"[1,1,1]"
```
