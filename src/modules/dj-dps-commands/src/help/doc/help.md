# help.help
Helps for command usage

### Aliases
They are follows aliases of command name: 
+ ```help```,  
+ ```h```.


### Command Parameters

+ ```direction```(aliased as ```dir```, ```for```): Direction of iteration (optional)


### How to use
```dps
help(command:'context')
```
